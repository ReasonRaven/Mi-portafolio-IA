Tarea 2

Escaneo de mi objeto

Escanee un koala con Polycam para convertirlo en un modelo 3D que despues puede visualizarse, compartirse o prepararse para impresion 3D.

Proceso

La aplicacion captura el objeto desde varios angulos y genera una malla 3D. El resultado se puede publicar en Sketchfab o usar como base para otros proyectos.
