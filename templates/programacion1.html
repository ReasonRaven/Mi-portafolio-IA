Juego del Ahorcado

C++

Ahorcado multidificultad con registro en archivo

Descripcion

Juego de ahorcado con 4 modos: Facil (palabras cortas), Medio (palabras largas), Dificil (frases de dos palabras) y 1 vs 1 donde un jugador ingresa la palabra y el otro adivina. Usa arreglos, funciones, manejo de archivos y numeros aleatorios.

  • Arreglos de strings
  • Funciones con prototipo
  • Manejo de archivos (fstream)
  • Numeros aleatorios (srand/rand)
  • Switch / do-while

        cout << "Palabra: " << palabraOculta << endl;
        mostrarAhorcado(vidas);

        char letra;
        cout << "Ingresa una letra: ";
        cin >> letra;

        bool encontrada = false;
        for (size_t i = 0; i < palabra.size(); i++)
            if (palabra[i] == letra) { palabraOculta[i] = letra; encontrada = true; }

Sensor de Temperatura y Presion

Programa que lee datos de temperatura, humedad y presion atmosferica, y emite alertas segun umbrales definidos. Usa condicionales anidados (if / else if).

// Act. 5 · Jonathan Hernandez Lazcano
#include <iostream>
using namespace std;

int temp = 0, humedad = 0, presion = 0;

int main() {
    cout << "Ingresa la temperatura: "; cin >> temp;
    cout << "Ingresa la humedad: ";     cin >> humedad;
    cout << "Ingrese la presion: ";     cin >> presion;

    if (temp > 30 && humedad > 60)
        cout << "Alerta alerta!!!";
    else if (temp >= 20 && temp <= 25 && humedad >= 40 && humedad <= 50)
        cout << "Las condiciones son ideales" << endl;

    if (presion <= 950 || presion >= 1050)
        cout << "Presion fuera de rango (950-1050 hPa)" << endl;
}

Calculadora de Ancho de Banda

Calcula el ancho de banda consumido por una plataforma de streaming (YouTube, Netflix, Twitch) en funcion del bitrate del video, la duracion y el numero de usuarios simultaneos.

// Act. 4 · Jonathan Hernandez Lazcano
// Caso de uso: plataformas de streaming de video
#include <iostream>
using namespace std;

double Bitrate = 0;
float  Duracion = 0;
int    Usuarios = 0;
float  result = 0;

int main() {
    cout << "Ingresa el Bitrate (Mbps): "; cin >> Bitrate;
    cout << "Ingresa la duracion (min): "; cin >> Duracion;
    cout << "Numero de usuarios:        "; cin >> Usuarios;

    // Formula: (Bitrate / 60) * (Duracion * 60) * Usuarios
    result = (Bitrate / 60) * (Duracion * 60) * Usuarios;

    cout << "El ancho de banda es: " << result << " MB" << endl;
}

Clase Persona

Ejemplo de clase con atributos privados (encapsulamiento), constructor y metodos publicos. Base para conceptos mas avanzados como Herencia y Polimorfismo que tambien se practicaron en el curso.

#include <iostream>
using namespace std;

class Persona {
private:                  // Atributos (encapsulados)
    int    edad;
    string nombre;

public:                         // Metodos (acciones)
    Persona(int, string);       // Constructor
    void leer();
    void correr();
};

// Constructor: inicializa los atributos
Persona::Persona(int _edad, string _nombre) {
    edad   = _edad;
    nombre = _nombre;
}

void Persona::leer() {
    cout << "Soy " << nombre << " y estoy leyendo" << endl;
}

void Persona::correr() {
    cout << "Soy " << nombre << " y estoy corriendo, tengo "
         << edad << " años" << endl;
}

int main() {
    Persona p1(19, "Jonathan");
    p1.leer();   // → Soy Jonathan y estoy leyendo
    return 0;
}

Serie de Fibonacci

Calculo de la serie de Fibonacci usando recursividad. La funcion se llama a si misma dos veces (caso general) hasta llegar al caso base (n < 2). Tambien se practicaron factorial, potencia y secuencias ascendentes/descendentes.

#include <iostream>
using namespace std;

int fibonacci(int n) {
    if (n < 2) return n;                       // Caso base
    return fibonacci(n - 1) + fibonacci(n - 2); // Caso recursivo
}

int main() {
    int n;
    cout << "Limite de la serie: ";
    cin >> n;

    if (n < 0)
        cout << "Ingresa un numero positivo";
    else
        cout << "Resultado: " << fibonacci(n);
}
